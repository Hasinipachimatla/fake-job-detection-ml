# Fake Job Posting Detection (ML Project)

## 📌 Overview
This project detects whether a job posting is **fake or real** using Machine Learning techniques.

## ⚙️ Technologies Used
- Python
- Scikit-learn
- NLP (Text Processing)
- TF-IDF Vectorization
- Logistic Regression

## 🚀 Features
- Data preprocessing (cleaning, stopwords removal)
- Feature extraction using TF-IDF
- Model training with Logistic Regression
- Handles imbalanced dataset using class_weight
- Custom input prediction

## 📊 Results
- Achieved good accuracy in detecting fake job postings
- Model can generalize to unseen job descriptions

## 🧪 Example
Input:
"Earn money fast with no experience"

Output:
Fake Job 🚨

## 📁 Files
- `fake_job_detection.ipynb` → Full project code
- `fake_job_model.pkl` → Trained model
- `vectorizer.pkl` → TF-IDF vectorizer
